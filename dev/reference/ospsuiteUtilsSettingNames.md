# Names of the settings stored in `ospsuiteEnv`. Can be used with `getOSPSuiteUtilsSetting()`

Names of the settings stored in `ospsuiteEnv`. Can be used with
[`getOSPSuiteUtilsSetting()`](https://www.open-systems-pharmacology.org/OSPSuite.RUtils/dev/reference/getOSPSuiteUtilsSetting.md)

## Usage

``` r
ospsuiteUtilsSettingNames
```

## Format

An object of class `list` of length 5.

## Examples

``` r
ospsuiteUtilsSettingNames
#> $packageName
#> [1] "packageName"
#> 
#> $suiteName
#> [1] "suiteName"
#> 
#> $logging
#> [1] "logging"
#> 
#> $formatNumericsDigits
#> [1] "formatNumericsDigits"
#> 
#> $LOG_SAFE_EPSILON
#> [1] "LOG_SAFE_EPSILON"
#> 
```
